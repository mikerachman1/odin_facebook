## Functionality
It was created to replicate all of the core functionality of Facebook including:
  - Profiles
  - Friends 
  - Posts
  - Likes
  - Image uploads
  - Notifications

This project also utilizes:
  - Action Mailer
  - Google omini auth login 
  - Devise
  - Turbo
